
# REST API GreenWay

This is REST API for GreenWay App.



## How to run on local

Clone this repository (branch-main)

```bash
  git clone https://github.com/AjengAlyaK/restapi-greenway.git
```
Go to the root folder

```bash
  cd yourfoldername
```

Install all dependency

```bash
  npm install
```

Run   
```bash
  npm run start
```
